# Connect The Pipes Game

A puzzle game where you connect pipes to complete the path and win.

## How to Run

1. Download or clone the repository
2. Open `Project_Game.sln` in **Visual Studio** (2019 or later recommended)
3. Set the configuration to **Debug x64** or **Release x64**
4. Press **F5** to build and run

> ⚠️ The [SGG library](https://github.com/cgaueb/sgg) is only supported on Windows via Visual Studio. No other IDEs or platforms are supported.
